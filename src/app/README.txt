the app folder stores all of the pages for the webapp
the main page is the Home.js page.

All other pages are added through link in the App.js
file. If you want to add a page, you have to import import
it in App.js and create the Link the same way all the 
other pages' links are created

Specifics on what these pages are meant to have are in 
the "Website necessities" google doc here
https://docs.google.com/document/d/1a2za7FOUS2c55QqXTBb3I9Xeg7kkGg7LB13pVHQHTGo/edit?usp=sharing
